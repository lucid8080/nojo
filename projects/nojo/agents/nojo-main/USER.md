# nojo-main — USER (what to ask)

Ask this agent when you want:

- coordination across Nojo agents
- a decomposition of a job into sub-tasks
- a final “ready-to-run” plan you can hand to a builder/support/sales/content agent

Suggested prompt format:

1. Goal (one sentence)
2. Constraints (time, compliance, tone)
3. What “good” looks like
4. Which Nojo agent(s) should own each part
