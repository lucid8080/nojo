# nojo-main — MEMORY (long-term notes)

This file is the long-term memory for the Nojo orchestration lead.

## Store

- The current Nojo project status summary (non-sensitive)
- Decisions made (with links/refs to `projects/nojo/DECISIONS.md` when applicable)
- Known constraints and “what worked” patterns for Nojo agent handoffs
- Cross-agent coordination notes that improve planning quality

## Do not store here

- Agent-specific personality drift rules (those belong in `IDENTITY.md` / `SOUL.md`)
- Private user personal memory
- Other agents’ long-term memory (keep it in their own `MEMORY.md`)

## Daily notes

Write incremental updates into `memory/<YYYY-MM-DD>.md`.
